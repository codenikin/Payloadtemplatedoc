### Upload the Project Files

<Steps>
1. Upload the project  files
   Upload the Project Files First, upload the botify.zip file to your VPS using FTP, SCP, or your preferred file transfer method
   
   ```js
   scp main.zip root@your-server-ip:/var/www/
   ```
2. Unzip the Project Once uploaded

    ```
    cd /var/www/
    sudo unzip botify.zip -d payloadcms

    ```

3.  Navigate to the Project Directory

    ```
    cd /var/www/payloadcms
    ```

4.  Update the .env File
    Open the `.env` file located in the root directory and update it with the correct credentials and configuration values

    ```
    DATABASE_URI=postgres://postgres:postgres@127.0.0.1:5432/postnew
    PAYLOAD_SECRET=726d8072d606413ddb914969
    GOOGLE_API_KEY=AIzaSyBxmm-V4MRg_cQxxz1rJC-9wGBKrt9-imI
    NEXT_PUBLIC_SERVER_URL=http://localhost:3000
    SMTP_HOST= smtp.hostinger.com
    SMTP_USER=sales@codenik.in
    SMTP_PASS=Codenik@2026


    ```

5.  Run the following command to install all project dependencies

              ```
              pnpm i
              ```

6.  Execute the initial database migrations using the command

                ```
                pnpm build
                ```

7.  Change directory to the database package:

                    ```
                    cd packages/db
                    ```

8.  Apply Prisma Migrations

                           ```
                           npx prisma migrate
                           ```

9.  You can now start the application using:

                   ```
                   pnpm start
                   ```

10. To keep your Node.js application running continuously in the background, use PM2:

                     ```
                     pm2 start pnpm -- start
                     ```

</Steps>

### Botify Installation with Docker

Note: Botify Docker installation requires an extended license. Please contact us for more details.

<Steps>

1. Upload the Project Files
   First, upload the botify.zip file to your VPS using FTP, SCP, or your preferred file transfer method

   ```js
   scp botify.zip root@your-server-ip:/var/www/

   ```

2. Unzip the Project
   Once uploaded

   ```
   cd /var/www/
   sudo unzip botify.zip -d botify

   ```

3. Navigate to the Project Directory

   ```
   cd /var/www/botify
   ```

4. Start the Application Using Docker

   ```
   sudo docker compose build up -d

   ```

5. Verify Containers are Running

   ```
   sudo docker ps

   ```

   You should see containers for:

   - botify-web (Next.js app)

   - redis

   - postgres

6. Access the Application

   ```
   http://yourdomain.com
   You will be automatically redirected to the Onboarding Page.

   ```

   <Badge text="Success" variant="success" />

   - Your Botify Bulk WhatsApp Sender is now successfully installed and running in a secure, Dockerized environment.

7. Create Admin Acoount

   ![Onboarding](/onboard1.png)

   From Here, you can create your first user account, which will be assigned administrator privileges. This admin account will allow you to manage the system, add Plans, and configure your WhatsApp sending settings.

</Steps>
