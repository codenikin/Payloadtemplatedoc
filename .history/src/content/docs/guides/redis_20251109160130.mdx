### Redis installation

<Steps>

1.  Install Redis with the following commands:

        ```js
        sudo apt update && sudo apt install redis-server -y


        ```

2.  Edit the Redis config file:

            ```
            sudo nano /etc/redis/redis.conf

            ```

</Steps>
