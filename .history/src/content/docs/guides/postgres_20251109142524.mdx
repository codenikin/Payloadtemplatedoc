### Upload the Project Files

<Steps>

1.  Install PostgreSQL with the following commands:

    ```js
    sudo apt update
    sudo apt install postgresql

    ```

2.  Edit the postgresql.conf file:

    ```
    cd /var/www/
    sudo unzip botify.zip -d botify

    ```

3.  Navigate to the Project Directory

    ```
    cd /var/www/botify
    ```
